Sport Gear Search Engine

AI-powered running shoe recommender using Strava activity data. Analyzes runner profile (cushioned, daily trainer, road, long run, etc.) and matches to running shoes.

Role: Full-stack: Built FastAPI backend, Random Forest multi-label classification, Playwright web crawler, TF-IDF search engine, responsive HTML/JS/CSS dashboard with pagination.

Technologies: Python, FastAPI, scikit-learn, Playwright, SQLite, HTML/CSS/JavaScript

Key Features:

  • login system
  • Strava OAuth login & data sync (distance, pace, cadence, HR, elevation)
  • Random Forest classifies runner style from real run data
  • Web crawler scrapes Nike shoes
