Sport Gear Search Engine

A personalized search engine for runners to find the most suitable Garmin running gear based on running type and preferences. The system classifies runners using machine learning and matches them with recommended products.

Role: Designed the front-end interface, implemented the search engine logic, integrated ML-based classification, and visualized the results for user-friendly browsing.

Technologies: Python, Machine Learning (scikit-learn), Web Crawling, HTML/CSS/JS

Key Features:

  • Classifies runner types based on fetching sport watch data
  • Search and filter gear recommendations using classified keywords
  • Interactive front-end for easy navigation
  • Backend logic for fetching, cleaning, and ranking products
